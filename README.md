Bank Management System - Tugas 2 PBO
Sistem perbankan sederhana ini dirancang untuk memenuhi kriteria Tugas 2 mata kuliah Pemrograman Berorientasi Objek dengan fokus utama pada penerapan prinsip Encapsulation. Arsitektur sistem ini terbagi menjadi tiga komponen utama: kelas BankAccount yang berfungsi menyimpan data entitas nasabah seperti nomor akun, nama pemilik, saldo, email, dan nomor telepon; kelas BankService yang menangani seluruh logika bisnis transaksi melalui metode deposit(), withdraw(), dan checkBalance(); serta kelas Main sebagai entry point aplikasi untuk menjalankan alur program. Implementasi Encapsulation dilakukan secara ketat dengan mengubah seluruh atribut menjadi private fields untuk melindungi data sensitif seperti saldo agar tidak dimodifikasi secara ilegal dari luar kelas. Akses terhadap data tersebut dikendalikan sepenuhnya melalui metode Getter dan Setter yang menyertakan logika validasi internal guna memastikan setiap transaksi tetap aman dan konsisten sesuai aturan sistem perbankan.
